# Mascot Assets

This directory should contain the mascot character images for each evolution stage:

## Required Files

- `stage0-egg.png` - Egg stage (0 XP)
- `stage1-chick.png` - Chick stage (100 XP)
- `stage2-young.png` - Young Chicken stage (500 XP)
- `stage3-smart.png` - Smart Chicken stage (1,500 XP)
- `stage4-wise.png` - Wise Rooster stage (5,000 XP)

## Design Specifications

### General Requirements
- **Format**: PNG with transparency
- **Size**: 300x300 pixels (will be scaled down in UI)
- **Style**: Consistent across all stages
- **Color Scheme**: Warm colors that complement the purple gradient UI

### Stage Characteristics

1. **Egg (stage0-egg.png)**
   - Simple egg shape
   - Maybe some cracks as XP increases (future enhancement)
   - Soft pastel colors

2. **Chick (stage1-chick.png)**
   - Small, cute baby chicken
   - Large eyes, small beak
   - Fluffy appearance
   - Maybe wearing a simple hat (unlockable)

3. **Young Chicken (stage2-young.png)**
   - Slightly larger, more defined features
   - Could have a book or pencil accessory
   - More alert/focused expression

4. **Smart Chicken (stage3-smart.png)**
   - Adult chicken
   - Wearing glasses or graduation cap
   - Confident posture
   - More detailed feathers

5. **Wise Rooster (stage4-wise.png)**
   - Majestic rooster
   - Crown or elaborate accessories
   - Impressive tail feathers
   - Wise, serene expression
   - Maybe a cape or other royal accessories

## Animation Ideas (Future)

- Idle animations (subtle breathing, blinking)
- Celebration animations (when challenge completed)
- Evolution animations (transformation between stages)
- Interaction animations (clicking on mascot)

## Customization Assets (Future)

Additional accessories to unlock:
- Hats: basic, scholar, graduation, crown
- Accessories: glasses, book, cape, wand
- Backgrounds: sky, galaxy, library, dojo
- Expressions: happy, focused, celebrating, sleeping

## Design Resources

Consider these art styles:
- Flat design (modern, clean)
- Cartoon style (friendly, approachable)
- Pixel art (retro, charming)
- Vector art (scalable, professional)

## Notes for Artist

- The mascot should be friendly and non-intimidating
- Should appeal to a wide age range (teens to adults)
- Evolution should show clear progression
- Each stage should feel rewarding to unlock
- Mascot is central to the user experience - invest time here!
